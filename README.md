# Aplikasi Keuangan Pribadi (Personal Finance Manager)

Project Akhir Praktikum Pemrograman Berorientasi Objek (PBO) Semester 3.
Aplikasi desktop berbasis Java Swing untuk mengelola keuangan pribadi secara efisien dengan tampilan modern.

## Fitur Utama
- **Dashboard Interaktif**: Ringkasan saldo, grafik pemasukan/pengeluaran, dan shortcut cepat.
- **Pencatatan Transaksi**: Catat pemasukan dan pengeluaran dengan kategori dan akun berbeda.
- **Manajemen Rekening**: Kelola berbagai dompet/rekening (Tunai, Bank, E-Wallet).
- **Budgeting**: Atur batas anggaran bulanan per kategori agar tidak boros.
- **Target Tabungan (Goals)**: Tetapkan tujuan menabung (misal: beli HP, liburan) dan pantau progresnya.
- **Laporan & Analisis**: Visualisasi data keuangan menggunakan diagram (Pie Chart & Bar Chart).

## Cara Menjalankan Aplikasi
1. Pastikan JDK (Java Development Kit) sudah terinstal (minimal versi 8).
2. Buka project menggunakan IDE (VS Code / IntelliJ / NetBeans).
3. Jalankan file utama: `src/AplikasiKeuanganPribadi.java`.

## Struktur Project
- `backend`: Logika bisnis, model data, dan services.
- `frontend`: Antarmuka pengguna (UI) menggunakan Java Swing.
- `data`: Penyimpanan data JSON (transaksi, akun, budget, goals).
